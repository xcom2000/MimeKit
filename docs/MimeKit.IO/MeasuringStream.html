MeasuringStream Class

A stream useful for measuring the amount of data written.

Syntax

public class MeasuringStream : System.IO.Stream

Remarks

A MimeKit.IO.MeasuringStream keeps track of the number of bytes that have been written to it. This is useful, for example, when you need to know how large a MimeKit.MimeMessage is without actually writing it to disk or into a memory buffer.

Write Method

Writes a sequence of bytes to the stream and advances the current position within this stream by the number of bytes written.

Syntax

public override void Write (byte[] buffer, int offset, int count)

Parameters

buffer
The buffer to write.
offset
The offset of the first byte to write.
count
The number of bytes to write.

Exceptions

Type Reason
ObjectDisposedException The stream has been disposed.
NotSupportedException The stream does not support writing.
ArgumentNullException buffer is null.
ArgumentOutOfRangeException

offset is less than zero or greater than the length of buffer.

-or-

The buffer is not large enough to contain count bytes starting at the specified offset.
System.IO.IOException An I/O error occurred.

Remarks

Increments the MeasuringStream.Position property by the number of bytes written. If the updated position is greater than the current length of the stream, then the MeasuringStream.Length property will be updated to be identical to the position.
