BoundStream Class

A bounded stream, confined to reading and writing data to a limited subset of the overall source stream.

Syntax

public class BoundStream : System.IO.Stream

Remarks

Wraps an arbitrary stream, limiting I/O operations to a subset of the source stream. If the BoundStream.EndBoundary is -1, then the end of the stream is unbound.

When a MimeKit.MimeParser is set to parse a persistent stream, it will construct MimeKit.ContentObjects using bounded streams instead of loading the content into memory.
