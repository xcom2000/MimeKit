MemoryBlockStream Class

An efficient memory stream implementation that sacrifices the ability to get access to the internal byte buffer in order to drastically improve performance.

Syntax

public class MemoryBlockStream : System.IO.Stream

Remarks

Instead of resizing an internal byte array, the MimeKit.IO.MemoryBlockStream chains blocks of non-contiguous memory. This helps improve performance by avoiding unneeded copying of data from the old array to the newly allocated array as well as the zeroing of the newly allocated array.
