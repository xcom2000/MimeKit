ApplicationPgpSignature Class

A MIME part with a Content-Type of application/pgp-signature.

Syntax

public class ApplicationPgpSignature : MimeKit.MimePart

Remarks

An application/pgp-signature part contains detatched pgp signature data and is typically contained within a MimeKit.Cryptography.MultipartSigned part.

To verify the signature, use the MultipartSigned.Verify method on the parent multipart/signed part.
