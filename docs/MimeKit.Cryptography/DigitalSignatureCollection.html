DigitalSignatureCollection Class

A collection of digital signatures.

Syntax

public class DigitalSignatureCollection : System.Collections.ObjectModel.ReadOnlyCollection<IDigitalSignature>

Remarks

When verifying a digitally signed MIME part such as a MimeKit.Cryptography.MultipartSigned or a MimeKit.Cryptography.ApplicationPkcs7Mime, you will get back a collection of digital signatures. Typically, a signed message will only have a single signature (created by the sender of the message), but it is possible for there to be multiple signatures.
