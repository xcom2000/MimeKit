SqliteCertificateDatabase Class

An X.509 certificate database built on SQLite.

Syntax

public class SqliteCertificateDatabase : SqlCertificateDatabase

Remarks

An X.509 certificate database is used for storing certificates, metdata related to the certificates (such as encryption algorithms supported by the associated client), certificate revocation lists (CRLs), and private keys.

This particular database uses SQLite to store the data.

Remarks

Creates a new MimeKit.Cryptography.SqliteCertificateDatabase and opens a connection to the SQLite database at the specified path using the Mono.Data.Sqlite binding to the native SQLite library.

If Mono.Data.Sqlite is not available or if an alternative binding to the native SQLite library is preferred, then consider using SqliteCertificateDatabase.#ctor(System.Data.IDbConnection, string) instead.
