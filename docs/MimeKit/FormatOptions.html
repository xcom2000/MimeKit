HiddenHeaders Property

Gets the message headers that should be hidden.

Syntax

public HashSet<HeaderId> HiddenHeaders { get; }

Value

The message headers.

Remarks

Specifies the set of headers that should be removed when writing a MimeKit.MimeMessage to a stream.

This is primarily meant for the purposes of removing Bcc and Resent-Bcc headers when sending via a transport such as SMTP.
