MimeKit

A C# MIME creation and parser library with support for S/MIME, PGP and Unix mbox spools.

MimeKit Namespace

Namespace

This is the top-level MimeKit namespace, where you'll find most of the types that you'll routinely be using.

Type Description
AttachmentCollection A collection of attachments.
BodyBuilder A message body builder.
ContentDisposition A class representing a Content-Disposition header value.
ContentEncoding An enumeration of all supported content transfer encodings. .
ContentObject Encapsulates a content stream used by MimeKit.MimePart.
ContentType A class representing a Content-Type header value.
DomainList A domain list.
EncodingConstraint A content encoding constraint.
FormatOptions Format options for serializing various MimeKit objects.
GroupAddress An address group, as specified by rfc0822.
Header A class representing a Message or MIME header.
HeaderId An enumeration of common header fields.
HeaderList A list of MimeKit.Headers.
HeaderListChangedAction Header list changed action.
IContentObject An interface for content stream encapsulation as used by MimeKit.MimePart.
InternetAddress An internet address, as specified by rfc0822.
InternetAddressList A list of MimeKit.InternetAddresses.
MailboxAddress A mailbox address, as specified by rfc0822.
MessageIdList A list of Message-Ids, as found in the References header.
MessagePart A MIME part containing a MimeKit.MimeMessage as its content.
MessagePartial A MIME part containing a partial message as its content.
MimeEntity An abstract MIME entity.
MimeEntityConstructorInfo MIME entity constructor info.
MimeFormat The format of the MIME stream.
MimeMessage A MIME message.
MimeParser A MIME parser which can be used to parse MimeKit.MimeMessages and MimeKit.MimeEntitys from arbitrary streams.
MimePart A basic leaf-node MIME part that contains content such as the message body or an attachment.
Multipart A multipart MIME entity which may contain a collection of MIME entities.
NewLineFormat A New-Line format.
Parameter A header parameter as found in the Content-Type and Content-Disposition headers.
ParameterList A list of parameters, as found in the Content-Type and Content-Disposition headers.
ParseException A Parse exception as thrown by the various Parse methods in MimeKit.
ParserOptions Parser options as used by MimeKit.MimeParser as well as various Parse and TryParse methods in MimeKit.
TextPart A Textual MIME part.
