InternetAddress Class

An internet address, as specified by rfc0822.

Syntax

public abstract class InternetAddress

Remarks

Serves as the base class for MimeKit.MailboxAddress and MimeKit.GroupAddress.

ToString Method

Serializes the MimeKit.InternetAddress to a string, optionally encoding it for transport.

Syntax

public abstract string ToString (bool encode)

Parameters

encode
If set to true, the MimeKit.InternetAddress will be encoded.

Returns

A string representing the MimeKit.InternetAddress.

Remarks

If the encode parameter is true, then this method will return an encoded version of the internet address according to the rules described in rfc2047.

However, if the encode parameter is false, then this method will return a string suitable only for display purposes.
